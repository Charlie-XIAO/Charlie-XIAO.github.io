Overview

This is a simple SQL engine supporting CSV data searching, various SQL operators, rule-based and cost-based optimization, and different modes of shuffling. Basic funtionality include:

  • Parsing and analyzing SQL to form an abstract syntax tree.
  • Supporting various SQL operators including Scan, Project, Filter, Aggregate, Limit, and different modes of Join. This functionality is not fully tested due to limitation of time and may contain bugs.
  • Supporing various rule-based optimization including predicate pushdown, limit pushdown, constant folding, column pruning, etc.
  • Supporting various cost-based optimization including join reorder, hash optimization, shuffle optimization, etc. This functionality is not fully implemented due to limitation of time.
