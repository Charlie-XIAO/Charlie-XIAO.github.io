Problem 1.1 - Floating Point

Julia supports $16$, $32$, and $64$ bit floating point numbers (i.e., half, single, and double precision). See here for details. Write a code to calculate machine precision for each of these types.
Solution. The machine precision is reached when the machine can no longer tell $1.0$ from its next floating point number. Therefore, we write the following code for printing out the machine precision of specific data types. 
function machep(dtype)
    """Print out the machine precision of the specified data type."""
    for i = 1 : 100
        macheps = 2.0 ^ (-i)

        end
    end
end

Solution. We implement the bisection method as follows. 
function bisection(func, left, right, tolerance)
    """Bisection method.

    Parameters
    ----------
    func : function
        The function to find a root for.
    left : Float64, etc.
        The left boundary of the interval to find a root.
    right : Float64, etc.
        The right boundary of the interval to find a root.
    tolerance : Float64, etc.
        The tolerance for the value of the root.

    Returns
    -------
    root : Float64 or boolean
        If a root is successfully found within tolerance, then it will be returned.
        Otherwise, the function returns false.
    """
    mid = (left + right) / 2
    mid_val = func(mid)
    if (abs(mid_val) < tolerance)
        return mid
    elseif (right - left < 1e-20)
        return false
    elseif (mid_val * func(left) < 0)
        return bisection(func, left, mid, tolerance)
    else
        return bisection(func, mid, right, tolerance)
    end
end

function newton(f, f_, x0, tolerance, maxiter)
    """Newton's method.

    Parameters
    ----------
    f : function
        The function to find a root for.
    f_ : function
        The derivative of the function to find a root for.
    x0 : Float64, etc.
        The initial guess of the root, better near the real root.
    tolerance : Float64, etc.
        The tolerance for the value of the root.
    maxiter : Int64, etc.
        The maximum number of iterations to run Newton's method.

    Returns
    -------
    root : Float64
        The root found by Newton's method.
    """
    cur_x, cur_f = x0, f(x0)
    for _ = 1 : maxiter
        next_x = cur_x - cur_f / f_(cur_x)
        if (abs(next_x - cur_x) < tolerance)
            break
        end
        cur_x, cur_f = next_x, f(next_x)
    end
    return cur_x
end

function newton_hist(f, f_, x0, tolerance, maxiter)
    """Newton's method with iteration history.

    Parameters
    ----------
    f : function
        The function to find a root for.
    f_ : function
        The derivative of the function to find a root for.
    x0 : Float64, etc.
        The initial guess of the root, better near the real root.
    tolerance : Float64, etc.
        The tolerance for the value of the root.
    maxiter : Int64, etc.
        The maximum number of iterations to run Newton's method.

    Returns
    -------
    x_hist : 1D array
        The history of the x values during the Newton's method.
    f_list : 1D array
        The history of the f(x) values during the Newton's method.
    """
    x_hist, f_hist = [], []
    cur_x, cur_f = x0, f(x0)
    for _ = 1 : maxiter
        next_x = cur_x - cur_f / f_(cur_x)
        if (abs(next_x - cur_x) < tolerance)
            break
        end
        push!(x_hist, cur_x)
        push!(f_hist, cur_f)
        cur_x, cur_f = next_x, f(next_x)
    end
    return x_hist, f_hist
end;

Problem 1.5 - Secant Convergence

In this problem, you will prove the rate of convergence for the secant method.
(a) Show that the secant method $$ \begin{align*} x_{k+1}=x_k-\frac{x_k-x_{k-1}}{f(x_k)-f(x_{k-1})}f(x_k) \end{align*} $$ can be rewritten in the form $$ \begin{align*} x_{k+1}=\frac{x_kf(x_{k-1})-x_{k-1}f(x_k)}{f(x_{k-1})-f(x_k)}. \end{align*} $$
Proof. We can write that $$ \begin{align*} x_{k+1}=x_k-\frac{x_k-x_{k-1}}{f(x_k)-f(x_{k-1})}f(x_k)={} & \frac{x_k(f(x_k)-f(x_{k-1}))-f(x_k)(x_k-x_{k-1})}{f(x_k)-f(x_{k-1})} \nonumber\\ ={} & \frac{x_{k-1}f(x_k)-x_kf(x_{k-1})}{f(x_k)-f(x_{k-1})}=\frac{x_kf(x_{k-1})-x_{k-1}f(x_k)}{f(x_{k-1})-f(x_k)}, \end{align*} $$ which gives the desired result.
(b) Now, denote the root of $f$ to be $\xi$, so that $f(\xi)=0$. Also assume that $f$ is twice continuously differentiable and that $f'>0$ and $f''>0$ in a neighborhoood of $\xi$. Define the quantity $\phi$ to be $$ \begin{align*} \phi(x_k,x_{k-1})=\frac{x_{k+1}-\xi}{(x_k-\xi)(x_{k-1}-\xi)}. \end{align*} $$ Compute (for fixed value of $x_{k-1}$) that $$ \begin{align*} \psi(x_{k-1})=\lim_{x_k\rightarrow\xi}\phi(x_k,x_{k-1}). \end{align*} $$
Solution. We can compute that $$ \begin{align*} \psi(x_{k-1})={} & \lim_{x_k\rightarrow\xi}\phi(x_k,x_{k-1})=\lim_{x_k\rightarrow\xi}\frac{x_{k+1}-\xi}{(x_k-\xi)(x_{k-1}-\xi)}=\lim_{x_k\rightarrow\xi}\frac{\frac{x_kf(x_{k-1})-x_{k-1}f(x_k)}{f(x_{k-1})-f(x_k)}-\xi}{(x_k-\xi)(x_{k-1}-\xi)} \nonumber\\ ={} & \lim_{x_k\rightarrow\xi}\frac{x_kf(x_{k-1})-x_{k-1}f(x_k)-\xi f(x_{k-1})+\xi f(x_k)}{(f(x_{k-1})-f(x_k))(x_k-\xi)(x_{k-1}-\xi)} \nonumber\\ ={} & \lim_{x_k\rightarrow\xi}\frac{(x_k-\xi)f(x_{k-1})-(x_{k-1}-\xi)f(x_k)}{(f(x_{k-1})-f(x_k))(x_k-\xi)(x_{k-1}-\xi)} \nonumber\\ ={} & \frac{f(x_{k-1})}{x_{k-1}-\xi}\cdot\lim_{x_k\rightarrow\xi}\frac{1}{f(x_{k-1})-f(x_k)}-\lim_{x_k\rightarrow\xi}\frac{1}{f(x_{k-1})-f(x_k)}\cdot\lim_{x_k\rightarrow\infty}\frac{f(x_k)}{x_k-\xi} \nonumber\\ ={} & \frac{f(x_{k-1})}{x_{k-1}-\xi}\cdot\frac{1}{f(x_{k-1})-f(\xi)}-\frac{1}{f(x_{k-1})-f(\xi)}\cdot\lim_{x_k\rightarrow\infty}\frac{f(x_k)-f(\xi)}{x_k-\xi} \nonumber\\ ={} & \frac{1}{x_{k-1}-\xi}-\frac{f'(\xi)}{f(x_{k-1})-f(\xi)}=\frac{f(x_{k-1})-x_{k-1}f'(\xi)+\xi f'(\xi)}{(x_{k-1}-\xi)f(x_{k-1})}. \end{align*} $$
(c) Now compute $$ \begin{align*} \lim_{x_{k-1}\rightarrow\xi}\psi(x_{k-1}), \end{align*} $$ and therefore show that $$ \begin{align*} \lim_{x_k,x_{k-1}\rightarrow\xi}\phi(x_k,x_{k-1})=\frac{f''(\xi)}{2f'(\xi)}. \end{align*} $$
Proof. We can compute that $$ \begin{align*} \lim_{x_{k-1}\rightarrow\xi}\psi(x_{k-1})={} & \lim_{x_{k-1}\rightarrow\xi}\frac{f(x_{k-1})-x_{k-1}f'(\xi)+\xi f'(\xi)}{(x_{k-1}-\xi)f(x_{k-1})}\overset{\text{L'Hosp.}}{=}\lim_{x_{k-1}\rightarrow\xi}\frac{f'(x_{k-1})-f'(\xi)}{f(x_{k-1})+(x_{k-1}-\xi)f'(x_{k-1})} \nonumber\\ \overset{\text{L'Hosp.}}{=}{} & \lim_{x_{k-1}\rightarrow\xi}\frac{f''(x_{k-1})}{f'(x_{k-1})+f'(x_{k-1})+(x_{k-1}-\xi)f''(x_{k-1})}=\frac{f''(\xi)}{2f'(\xi)}. \end{align*} $$ Therefore, we can conclude that $$ \begin{align*} \lim_{x_k,x_{k-1}\rightarrow\xi}\phi(x_k,x_{k-1})=\lim_{x_{k-1}\rightarrow\xi}\psi(x_{k-1})=\frac{f''(\xi)}{2f'(\xi)}, \end{align*} $$ and the proof is done.
(d) Next, assume that the secant method has convergence order $q$, that is to say that $$ \begin{align*} \lim_{k\rightarrow\infty}\frac{|x_{k+1}-\xi|}{|x_k-\xi|^q}=A<\infty. \end{align*} $$ Using the above results, show that $q-1-1/q=0$, and therefore that $q=(1+\sqrt{5})/2$.
Proof. Assuming that the secant method has convergence order $q$, we have that $$ \begin{align*} \lim_{k\rightarrow\infty}|x_{k+1}-\xi|=A\lim_{k\rightarrow\infty}|x_k-\xi|^q\implies\lim_{k\rightarrow\infty}\ln|x_{k+1}-\xi|=\ln A+q\lim_{k\rightarrow\infty}\ln|x_k-\xi|. \end{align*} $$ Denote $\epsilon_{t}:=\lim_{k\rightarrow\infty}\ln|x_t-\xi|$, $t\in\set{k-1,k,k+1}$, then we can rewrite and deduce that $$ \begin{align*} \epsilon_{k+1}=\ln A+q\epsilon_k,\quad\epsilon_k=\ln A+q\epsilon_{k-1}\implies\epsilon_{k-1}=\frac{\epsilon_k-\ln A}{q}. \end{align*} $$ Bringing in this result, now we can compute that $$ \begin{align*} \lim_{k\rightarrow\infty}\ln\phi(x_k,x_{k-1})=\epsilon_{k+1}-\epsilon_k-\epsilon_{k-1}=\bkt{1+\frac{1}{q}}\ln A+\bkt{q-1-\frac{1}{q}}\epsilon_k. \end{align*} $$ Note that $\epsilon_k$ is (negative) infinity, since $x_k\rightarrow\xi$ as $k\rightarrow\infty$. However, the left-hand side is finite, which can be deduced from the limit. Therefore, we must require $$ \begin{align*} q-1-\frac{1}{q}=0\implies q^2-q-1=0\implies q=\frac{1+\sqrt{5}}{2}\quad\text{(ignoring negative solution)}. \end{align*} $$ This is exactly the desired result.
(e) Finally, show that this implies that $$ \begin{align*} \lim_{k\rightarrow\infty}\frac{|x_{k+1}-\xi|}{|x_k-\xi|^q}=\bkt{\frac{f''(\xi)}{2f'(\xi)}}^{q/(1+q)}. \end{align*} $$
Proof. Using the limit and the previous result on $q$, we can rewrite that $$ \begin{align*} \ln\bkt{\frac{f''(\xi)}{2f'(\xi)}}=\bkt{1+\frac{1}{q}}\ln A\implies\ln A=\frac{q}{q+1}\cdot\ln\bkt{\frac{f''(\xi)}{2f'(\xi)}}\implies A=\bkt{\frac{f''(\xi)}{2f'(\xi)}}^{q/(1+q)}. \end{align*} $$ Therefore, we can conclude that $$ \begin{align*} \lim_{k\rightarrow\infty}\frac{|x_{k+1}-\xi|}{|x_k-\xi|^q}=A=\bkt{\frac{f''(\xi)}{2f'(\xi)}}^{q/(1+q)}, \end{align*} $$ which proves the desired result.

Problem 1.6 - Vector and Matrix Norm Inequalities

(a) Let $\mathbf{v}=(v_1,\cdots,v_n)^\top\in\RR^n$. Prove the following two inequalities: $$ \begin{align*} \norm{\mathbf{v}}_{\infty}\leq{} & \norm{\mathbf{v}}_2, \\ \norm{\mathbf{v}}_2^2\leq{} & \norm{\mathbf{v}}_1\norm{\mathbf{v}}_{\infty}. \end{align*} $$ For each inequality, given an example of a non-zero vector $\mathbf{v}$ for which equality is obtained. Finally, show that $$ \begin{align*} \norm{\mathbf{v}}_{\infty}\leq{} & \norm{\mathbf{v}}_2\leq\norm{\mathbf{v}}_1, \\ \norm{\mathbf{v}}_2\leq{} & \sqrt{n}\norm{\mathbf{v}}_{\infty}. \end{align*} $$
Proof. For the first inequality, we have that $$ \begin{align*} \norm{\mathbf{v}}_{\infty}=\max_{1\leq i\leq n}|v_i|=\max_{1\leq i\leq n}\sqrt{|v_i|^2}=\sqrt{\max_{1\leq i\leq n}|v_i|^2}\leq\sqrt{\sum_{i=1}^n|v_i|^2}=\norm{\mathbf{v}}_2, \end{align*} $$ where the equality holds when $\mathbf{v}$ has at most one non-zero component, for instance, $\mathbf{v}=(1,0,\cdots,0)^\top$. For the second inequality, we have that $$ \begin{align*} \norm{\mathbf{v}}_2^2=\sum_{i=1}^n|v_i|^2=\sum_{i=1}^n|v_i|\bkt{\max_{1\leq i\leq n}|v_i|}=\max_{1\leq i\leq n}|v_i|\cdot\sum_{i=1}^n|v_i|=\norm{\mathbf{v}}_{\infty}\norm{\mathbf{v}}_1, \end{align*} $$ where the equality holds when all components of $\mathbf{v}$ are equal, for instance, $\mathbf{v}=(1,\cdots,1)^\top$. For the third inequality, the left-hand side has already been shown, so it suffices to prove the right-hand side. Indeed, we have that $$ \begin{align*} \norm{\mathbf{v}}_2=\sqrt{\sum_{i=1}^n|v_i|^2}\leq\sqrt{\sum_{i=1}^n|v_i|^2+\sum_{i\lt j}|v_i||v_j|}=\sqrt{\bkt{\sum_{i=1}^n|v_i|}^2}=\sum_{i=1}^n|v_i|=\norm{\mathbf{v}}_1, \end{align*} $$ where the equality holds when $\mathbf{v}=\mathbf{0}$. Finally, for the last inequality, we can show that $$ \begin{align*} \norm{\mathbf{v}}_2=\sqrt{\sum_{i=1}^n|v_i|^2}\leq\sqrt{\sum_{i=1}^n\bkt{\max_{1\leq j\leq n}|v_i|}^2}=\sqrt{n\bkt{\max_{1\leq j\leq n}|v_i|}^2}=\sqrt{n}\max_{1\leq j\leq n}|v_i|=\sqrt{n}\norm{\mathbf{v}}_{\infty}, \end{align*} $$ where the equality holds when all components of $\mathbf{v}$ are equal, for instance, $\mathbf{v}=(1,\cdots,1)^\top$.
(b) Next, let $A\in\RR^{m\times n}$ be any real-valued matrix. Show that $$ \begin{align*} \norm{A}_{\infty}\leq{} & \sqrt{n}\norm{A}_2, \\ \norm{A}_2\leq{} & \sqrt{m}\norm{A}_{\infty}, \end{align*} $$ where above, the norms are interpreted as the induced matrix norms. For each inequality, give an example matrix $A$ for which equality is attained.
Proof. For the first inequality, we can deduce that $$ \begin{align*} \norm{A}_{\infty}=\max_{\mathbf{u}\in\RR^n}\frac{\norm{A\mathbf{u}}_{\infty}}{\norm{\mathbf{u}}_{\infty}}\leq\max_{\mathbf{u}\in\RR^n}\frac{\norm{A\mathbf{u}}_2}{\norm{\mathbf{u}}_2/\sqrt{n}}=\sqrt{n}\max_{\mathbf{u}\in\RR^n}\frac{\norm{A\mathbf{u}}_2}{\norm{\mathbf{u}}_2}=\sqrt{n}\norm{A}_2, \end{align*} $$ where an example of $A$ such that the equality holds is a matrix with the first row all $1$'s and the other entries all $0$'s. Problem 1.7 - Frobenius Matrix Norm

Another commonly used matrix norm in linear algebra is the Frobenius norm, defined for a matrix $A\in\RR_{m\times n}$ as $$ \begin{align*} \norm{A}_F=\sqrt{\sum_{i=1}^m\sum_{j=1}^n|a_{ij}|^2}. \end{align*} $$
(a) Prove that $\norm{A}_F=\sqrt{\tr(A^\top A)}$.
Proof. We can compute that $$ \begin{align*} A^\top A= \begin{pmatrix} a_{11} & \cdots & a_{m1} \\ \vdots & \ddots & \vdots \\ a_{1n} & \cdots & a_{mn} \end{pmatrix} \begin{pmatrix} a_{11} & \cdots & a_{1n} \\ \vdots & \ddots & \vdots \\ a_{m1} & \cdots & a_{mn} \end{pmatrix} = \begin{pmatrix} \sum_{i=1}^ma_{i1}^2 & \cdots & \sum_{i=1}^ma_{i1}a_{in} \\ \vdots & \ddots & \vdots \\ \sum_{i=1}^ma_{i1}a_{in} & \cdots & \sum_{i=1}^ma_{in}^2 \end{pmatrix}. \end{align*} $$ Therefore, the trace of $A^\top A$ can be computed as $$ \begin{align*} \tr(A^\top A)=\sum_{j=1}^n(A^\top A)_{jj}=\sum_{j=1}^n\sum_{i=1}^ma_{ij}^2=\sum_{i=1}^m\sum_{j=1}^n|a_{ij}|^2. \end{align*} $$ Therefore, by definition of the Frobenius norm, we can conclude that $$ \begin{align*} \norm{A}_F=\sqrt{\sum_{i=1}^m\sum_{j=1}^n|a_{ij}|^2}=\sqrt{\tr(A^\top A)}, \end{align*} $$ and the proof is complete.
(b) Prove also that $\norm{A}_F=\sqrt{\sum_j\sigma_j^2}$, where $\sigma_j$ are the singular values of $A$.
Proof. By singular value decomposition (for real matrices), we can write $$ \begin{align*} A=U\Sigma V^\top, \end{align*} $$ where $U\in\RR^{m\times m}$ and $V\in\RR^{n\times n}$ are unitary matrices and $\Sigma\in\RR^{m\times n}$ is a rectangular diagonal matrix with diagonal entries equal to the singular values of $A$. Therefore, we can expand $$ \begin{align*} \norm{A}_F=\sqrt{\tr(A^\top A)}=\sqrt{\tr(V\Sigma^\top U^\top U\Sigma V^\top)}=\sqrt{\tr(V\Sigma^\top\Sigma V^\top)}=\sqrt{\tr(\Sigma^\top\Sigma V^\top V)}=\sqrt{\tr(\Sigma^\top\Sigma)}, \end{align*} $$ where we make use of the property $\tr(MN)=\tr(NM)$ for any matrix $M$ and $N$, and that $M^\top M=I$ for any unitary real matrix $M$. Now note that $\Sigma^\top\Sigma$ is an $n\times n$ diagonal matrix with diagonal entries equal to the square of the singular values of $A$. Therefore, we can conclude that $$ \begin{align*} \norm{A}_F=\sqrt{\sum_{j=1}^n\sigma_j^2}, \end{align*} $$ where $\sigma_j$ are the singular values of $A$, and the proof is complete.
(c) Show that $\norm{A}_2\leq\norm{A}_F$.
Proof. For an arbitrary $\mathbf{x}\in\RR^n$, we can see that $$ \begin{align*} \norm{A\mathbf{x}}_2={} & \sqrt{\mathbf{x}^\top A^\top A\mathbf{x}}=\sqrt{\mathbf{x}^\top V\Sigma^\top U^\top U\Sigma V^\top\mathbf{x}}=\sqrt{\mathbf{x}^\top V\Sigma^\top\Sigma V^\top\mathbf{x}}=\sqrt{\tr(\mathbf{x}^\top V\Sigma^\top\Sigma V^\top\mathbf{x})} \nonumber\\ ={} & \sqrt{\tr(\mathbf{x}^\top\Sigma^\top\Sigma V^\top V\mathbf{x})}=\sqrt{\tr(\mathbf{x}^\top\Sigma^\top\Sigma\mathbf{x})}=\sqrt{\mathbf{x}^\top\Sigma^\top\Sigma\mathbf{x}}=\sqrt{\sum_{j=1}^n\sigma_j^2x_j^2}. \end{align*} $$ Also, note that $$ \begin{align*} \sum_{j=1}^n\sigma_j^2x_j^2\leq\sum_{j=1}^n\sigma_j^2x_j^2+\sum_{i\neq j}\sigma_ix_j=\bkt{\sum_{i=1}^n\sigma_i^2}\bkt{\sum_{i=1}^nx_i^2}. \end{align*} $$ Therefore, we can conclude that $$ \begin{align*} \norm{A}_2=\max_{\norm{\mathbf{x}}_2=1}\norm{A\mathbf{x}}_2={} & \max_{\norm{\mathbf{x}}_2=1}\sqrt{\sum_{j=1}^n\sigma_j^2x_j^2}\leq\max_{\norm{\mathbf{x}}_2=1}\sqrt{\bkt{\sum_{i=1}^n\sigma_i^2}\bkt{\sum_{i=1}^nx_i^2}} \nonumber\\ ={} & \max_{\norm{\mathbf{x}}_2=1}\sqrt{\sum_{i=1}^n\sigma_i^2}=\sqrt{\sum_{i=1}^n\sigma_i^2}=\norm{A}_F, \end{align*} $$ where the equality holds when all the singular values of $A$ are zero, and the proof is complete.

function classical_gs(n, A)
    """Classical Gram-Schmidt orthogonalization algorithm.

    Parameters
    ----------
    n : Int64, etc.
        The dimension of the matrix to be orthogonalized.
    A : Matrix{Float64}, etc.
        The matrix to be orthogonalized.

    Returns
    -------
    Q : Matrix{Float64}
        The orthogonalized matrix via the classical Gram-Schmidt algorithm.
    """
    Q = Array{Float64}(undef, n, n)
    for j = 1 : n
        v = copy(A[:, j])
        for i = 1 : (j - 1)
            v .-= dot(Q[:, i], A[:, j]) * Q[:, i]
        end
        Q[:, j] = v / norm(v)
    end
    return Q
end

However, as we have seen, Newton's method requires an initialization close to the root one wants to find, and it can be difficult to find all roots of a polynomial. Luckily, one can use the following relationship between eigenvalues and polynomial roots to find all roots of a given polynomial at once: Let us consider a polynomial of degree $n$ with leading coefficient $a_n=1$: $$ \begin{align*} p(x)=a_0+a_1x+\cdots+a_{n-1}x^{n-1}+x^n,\qquad\text{with $a_i\in\RR$}. \end{align*} $$
(a) Show that $p$ is the characteristic polynomial of the matrix $$ \begin{align*} A_p= \begin{pmatrix} 0 & & & -a_0 \\ 1 & \ddots & & -a_1 \\ & \ddots & 0 & \vdots \\ & & 1 & -a_{n-1} \end{pmatrix}\in\RR^{n\times n}, \end{align*} $$ which is sometimes called a companion matrix for $p$. Thus, the roots of $p$ can be computed as the eigenvalues of $A_p$ using, for example, the QR algorithm. Julia's eigen function can do this computation.
Solution. By cofactor expansion with respect to the first row, we have that $$ \begin{align*} & \det(A_p-xI)=\det \begin{pmatrix} -x & & & -a_0 \\ 1 & \ddots & & -a_1 \\ & \ddots & -x & \vdots \\ & & 1 & -a_{n-1}-x \end{pmatrix} \nonumber\\ & \qquad\qquad=-x\cdot\det \begin{pmatrix} -x & & & -a_1 \\ 1 & \ddots & & -a_2 \\ & \ddots & -x & \vdots \\ & & 1 & -a_{n-1}-x \end{pmatrix}+(-1)^{n+1}(-a_0)\cdot\det \underbrace{\begin{pmatrix} 1 & -x & & \\ 0 & \ddots & \ddots & \\ & \ddots & \ddots & -x \\ & & 0 & 1 \end{pmatrix}}_{\text{upper triangular}} \nonumber\\ & \qquad\qquad=-x\cdot\det \begin{pmatrix} -x & & & -a_1 \\ 1 & \ddots & & -a_2 \\ & \ddots & -x & \vdots \\ & & 1 & -a_{n-1}-x \end{pmatrix}+(-1)^na_0. \end{align*} $$ Similarly, we can write that $$ \begin{align*} \det \begin{pmatrix} -x & & & -a_1 \\ 1 & \ddots & & -a_2 \\ & \ddots & -x & \vdots \\ & & 1 & -a_{n-1}-x \end{pmatrix}=-x\det \begin{pmatrix} -x & & & -a_2 \\ 1 & \ddots & & -a_3 \\ & \ddots & -x & \vdots \\ & & 1 & -a_{n-1}-x \end{pmatrix}-(-1)^na_1. \end{align*} $$ Plugging this result into the previous expression, we can obtain that $$ \begin{align*} \det(A_p-xI)=x^2\det \begin{pmatrix} -x & & & -a_2 \\ 1 & \ddots & & -a_3 \\ & \ddots & -x & \vdots \\ & & 1 & -a_{n-1}-x \end{pmatrix}+(-1)^na_1x+(-1)^na_0. \end{align*} $$ Iterating this process, we can finally obtain that $$ \begin{align*} \det(A_p-xI)={} & (-x)^{n-2}\det \begin{pmatrix} -x & -a_{n-2} \\ 1 & -a_{n-1}-x \end{pmatrix}+(-1)^n(a_{n-3}x^{n-3}+\cdots+a_1x+a_0) \nonumber\\ ={} & (-x)^{n-2}(a_{n-1}x+x^2+a_{n-2})+(-1)^n(a_{n-3}x^{n-3}+\cdots+a_1x+a_0) \nonumber\\ ={} & (-1)^n(x^n+a_{n-1}x^{n-1}+a_{n-2}x^{n-2}+a_{n-3}x^{n-3}+\cdots+a_1x+a_0). \end{align*} $$ By keeping the leading coefficient as $1$, this is exactly $p(x)$. A more formal proof can be done via mathematical induction, but I think this way of writing it is more intrinsic.

function pwcoefs(n)
    """Returns the coefficients of Wilkinson's polynomial.

    Parameters
    ----------
    n : Int64, etc.
        The order of the Wilkinson's polynomial, i.e., the polynomial
        (x - 1)(x - 2)(x - 3) ... (x - n)
    
    Returns
    -------
    coeffs : Array{Float64}, etc.
        The array of coefficients, starting from the constant term.

    """
    coeffs = zeros(n + 1)
    coeffs[1] = 1
    for i in 1 : n
        for j in (i + 1) : -1 : 2
            coeffs[j] = coeffs[j - 1] - i * coeffs[j]
        end
        coeffs[1] = -i * coeffs[1]
    end
    return coeffs
end

Problem 3.3 - Matrix Functions

Let $A\in\CC^{n\times n}$ be a \textit{normal} matrix. For $t\in\RR$, the matrix exponential is defined as $$ \begin{align*} e^{tA}=I+tA+\frac{(tA)^2}{2!}+\frac{(tA)^3}{3!}+\cdots=\sum_{k=0}^{\infty}\frac{(tA)^k}{k!}. \end{align*} $$
(a) Can you relate the eigenvalue decomposition of $A$ to that of $e^{tA}$?
Solution. Suppose that $A$ can be decomposed as $$ \begin{align*} A=Q\Lambda Q^{-1}, \end{align*} $$ where $\Lambda$ is a diagonal matrix with diagonal entries the eigenvalues of $A$, and $Q$ is the matrix in which the $i$th column is the eigenvector corresponding to the $i$th eigenvalue in $\Lambda$. Then, we can compute that $$ \begin{align*} tA={} & Q\Lambda Q^{-1}, \\ (tA)^2={} & t^2Q\Lambda Q^{-1}Q\Lambda Q^{-1}=t^2Q\Lambda^2Q^{-1}, \\ \vdots \nonumber\\ (tA)^k={} & t^kQ\Lambda^kQ^{-1}. \end{align*} $$ Then, by definition of the matrix exponential, we have that $$ \begin{align*} e^{tA}=\sum_{k=0}^{\infty}\frac{t^kQ\Lambda^kQ^{-1}}{k!}=Q\bkt{\sum_{k=0}^{\infty}\frac{(t\Lambda)^k}{k!}}Q^{-1}=Qe^{t\Lambda}Q^{-1}. \end{align*} $$
(b) Now further suppose that the matrix $A$ is symmetric positive definite. If $X$ is a matrix such that $X^*X=A$, then $X$ is referred to as the square root of $A$. If you are given the eigendecomposition of $A$, i.e., $A=UDU^{-1}$, can you find the square root of $A$?
Solution. Denote the diagonal elements, i.e., the eigenvalues of $A$ as $\lambda_1,\cdots,\lambda_n$. Note that since $A$ is symmetric positive definite, all these eigenvalues are real and positive. Now consider the diagonal matrix $S$ with diagonal elements $\sqrt{\lambda_1},\cdots,\sqrt{\lambda_n}$. Then, taking $X=USU^{-1}$, we can compute that $$ \begin{align*} X^*X=\underbrace{(U^{-1})^*}_{=U}S^*\underbrace{U^*U}_{=I}SU^{-1}=US^*SU^{-1}=UDU^{-1}=A, \end{align*} $$ since $U$ is unitary due to the symmetry of $A$, and that $$ \begin{align*} S^*S= \begin{pmatrix} \sqrt{\lambda_1} & & \\ & \ddots & \\ & & \sqrt{\lambda_n} \end{pmatrix} \begin{pmatrix} \sqrt{\lambda_1} & & \\ & \ddots & \\ & & \sqrt{\lambda_n} \end{pmatrix}= \begin{pmatrix} \lambda_1 & & \\ & \ddots & \\ & & \lambda_n \end{pmatrix}=D. \end{align*} $$ Therefore, we can conclude that the square root of $A$ is $X=USU^{-1}$.
(c) Is the matrix square root unique? If not, provide an alternative square root matrix to the one in the previous part.
Solution. The matrix square root is not unique. Take $Y=SU^{-1}$, then we can check that $$ \begin{align*} Y^*Y=\underbrace{(U^{-1})^*}_{=U}S^*SU^{-1}=US^*SU^{-1}=UDU^{-1}. \end{align*} $$
(d) Motivated by the previous questions, can you determine a method for computing general matrix functions of real symmetric matrices? That is, if $f=f(x)$ is a scalar function, how can one define $f(A)$?
Solution. Using the same notations as the question above, we define $f(A)$ such that $$ \begin{align*} f(A)=U \begin{pmatrix} f(\lambda_1) & & \\ & \ddots & \\ & & f(\lambda_n) \end{pmatrix} U^{-1}. \end{align*} $$
(e) In class, we saw how to use Newton's method to find the square root of a number $a$ by finding the root of the function $f(x)=x^2-a$. Can you use a similar Newton iteration to find the square root of a symmetric positive definite matrix $A$ by finding the root of some matrix function $f=f(A)$? Implement your algorithm and use it to find the square root of the $6\times6$ matrix $A$ with entries $$ \begin{align*} A_{ij}=5\delta_{ij}+\sqrt{i^2+j^2}. \end{align*} $$
Solution. Analogous to Newton's method, we use a similar iteration such that $$ \begin{align*} X_{k+1}=X_k-(2X_k)^{-1}(X_k^2-A)=X_k-\frac{X_k-X_k^{-1}A}{2}=\frac{X_k+X_k^{-1}A}{2}. \end{align*} $$ We take the initial value $X_0=A$ and terminate when $\norm{X_k-X_{k-1}}\lt 10^{-12}$. The algorithm can thus be implemented as follows. 
function mat_sqrt(A)
    """Find the square root of a symmetric positive definite matrix.

    Parameters
    ----------
    A : Matrix{Float64}, etc.
        The matrix to find the square root of.
    
    Returns
    -------
    sqA : Matrix{Float64}
        The square root of the given matrix.
    """
    X = copy(A)
    for _ = 1 : 1000
        X_new = (inv(X) * A + X) / 2
        if norm(X_new - X) < 1e-12
            break
        end
        X = X_new
    end
    return X
end

Solution. We can implement Jacobi's method as follows. 
function _generate_Rpq(n, p, q, phi)
    """Generate R_pq for the Jacobi method."""
    R, c, s = Array{Float64}(I, n, n), cos(phi), sin(phi)
    R[p, p] = c
    R[q, q] = c
    R[p, q] = s
    R[q, p] = -s
    return R
end

function jacobi_eigen(n, A)
    """The Jacobi method for eigenproblem.
    
    Parameters
    ----------
    n : Int64, etc.
        The dimension of the matrix to be found eigenvalues and eigenvectors of.
    A : Matrix{Float64}, etc.
        The matrix to be found eigenvalues of.

    Returns
    -------
    eigvals : Array{Float64}
        The array of eigenvalues of the given matrix.
    eigvecs : Matrix{Float64}
        The matrix where each column is an eigenvector of the given matrix.

    Notes
    -----
    Note that the eigenvalues and the eigenvectors given in `eigvals` and `eigvecs`
    respectively are in the same order.
    """
    D, eigvecs, p, q, target = copy(A), Matrix{Float64}(I, n, n), 0, 0, -1
    for i = 1 : n
        for j = 1 : n
            if i != j && abs(D[i, j]) > target
                p, q, target = i, j, abs(D[i, j])
            end
        end
    end
    for _ = 1 : 1000
        phi = atan(2 * D[p, q] / (D[q, q] - D[p, p])) / 2
        R = _generate_Rpq(n, p, q, phi)
        D = R' * D * R
        eigvecs *= R
        target, sum_of_squares = -1, 0
        for i = 1 : n
            for j = 1 : n
                if i != j
                    sum_of_squares += D[i, j] ^ 2
                    if abs(D[i, j]) > target
                        p, q, target = i, j, abs(D[i, j])
                    end
                end
            end
        end
        if sqrt(sum_of_squares) <= 1e-8
            break
        end
    end
    return diag(D), eigvecs
end

Solution. The QR algorithm for finding singular values can be implemented as follows. 
function qr_svd(n, A)
    """The QR algorithm.

    Parameters
    ----------
    n : Int64, etc.
        The dimension of the matrix to find singular values of.
    A : Matrix{Float64}, etc.
        The matrix to find singular values of.
    
    Returns
    -------
    singvals : Array{Float64}
        The array of singular values of the given matrix.
    """
    B = A' * A
    prev = 1
    for _ = 1 : 1000
        (Q, R) = qr(B)
        B = R * Q
        sum_of_squares = 0
        for i = 2 : n
            for j = 1 : (i - 1)
                sum_of_squares += B[i, j] ^ 2
            end
        end
        if abs((sqrt(sum_of_squares) - prev) / prev) <= 1e-6
            break
        end
        prev = sum_of_squares
    end
    return sqrt.(diag(B))
end

        The point to evaluate the polynomial interpolant at.

    -------
    p : Float64
        The value of the polynomial interpolant at the given point.
    """
    p, xs = 0, [i / n - 1 for i in 0 : (2 * n)]

            else

Finally, the second Barycentric form can be implemented as follows. 
function barycentric2(f, n, x)
    """The second Barycentric form of polynomial interpolant.

    Parameters
    ----------
    f : Function of the form lambda x: f(x)
        The function to interpolate.
    n : Int64, etc.
        The degree of the polynomial interpolant.
    x : Float64, etc.
        The point to evaluate the polynomial interpolant at.
    
    Returns
    -------
    p : Float64
        The value of the polynomial interpolant at the given point.
    """
    xs = [i / n - 1 for i in 0 : (2 * n)]
    numerator, denominator = 0, 0
    for k = 0 : (2 * n)
        wk = 1
        for i = 0 : (2 * n)
            if i == k
                continue
            else
                wk /= (xs[k + 1] - xs[i + 1])
            end
        end
        numerator += f(xs[k + 1]) * wk / (x - xs[k + 1])
        denominator += wk / (x - xs[k + 1])
    end
    return numerator / denominator
end

(a) Write down the interpolating polynomial in Lagrange form of degree $1$ for the function $f(x)=x^3$ using the points $x_0=0$ and $x_1=a$.
Solution. We can compute that $$ \begin{align*} L_0(x)=\frac{x-x_1}{x_0-x_1}=-\frac{x-a}{a},\qquad L_1(x)=\frac{x-x_0}{x_1-x_0}=\frac{x}{a}. \end{align*} $$ Therefore, the interpolating polynomial of degree $1$ can be written as $$ \begin{align*} p_1(x)=L_0(x)f(x_0)+L_1(x)f(x_1)=-\frac{x-a}{a}\cdot0+\frac{x}{a}\cdot a^3=a^2x. \end{align*} $$
(b) Theorem 6.2 in Suli & Mayers states that for $x\in[0,a]$, there exists $\xi=\xi(x)\in(0,a)$, such that $$ \begin{align*} f(x)-p_n(x)=\frac{f^{(n+1)}(\xi)}{(n+1)!}\prod_{j=0}^n(x-x_j). \end{align*} $$ Here, $n$ is the degree of the interpolating polynomial $p_n$. Verify the above formula, by direct calculation, for the function and interpolating polynomial from (a). Show that in this case, $\xi$ is unique and has the value $\xi=\frac{1}{3}(x+a)$.
Proof. Here we have $n=1$. The second derivative of $f$ is thus $f^{(2)}(x)=6x$. Therefore, by taking $\xi=\frac{1}{3}(x+a)$, we have that $$ \begin{align*} \frac{f^{(2)}(\xi)}{2!}\prod_{j=0}^1(x-x_j)=\frac{2(x+a)}{2}\cdot x\cdot(x-a)=x(x+a)(x-a)=x^3-a^2x=f(x)-p_1(x), \end{align*} $$ as desired. Moreover, indeed $\xi=\frac{1}{3}(x+a)\in\sqbkt{\frac{a}{3},\frac{2a}{3}}\subseteq(0,a)$ given $x\in[0,a]$, so the existence is done. Now assume that there exists $\xi'\in(0,a)$ that satisfies the same equation and such that $\xi'\neq\xi$. This means that $$ \begin{align*} \frac{6\xi'}{2}\cdot x\cdot(x-a)=f(x)-p_1(x)=\frac{6\xi}{2}\cdot x\cdot(x-a)\implies(\xi'-\xi)x(x-a)=0. \end{align*} $$ By arbitrariness of $x$, we must have $\xi'=\xi$, leading to a contradiction. Therefore, the uniqueness of $\xi$ is also done, and the proof is complete.

Problem 4.4 - Chebyshev Polynomials

There are two commonly used equivalent formulas defining Chebyshev polynomials, which are $$ \begin{align*} T_n(x)=\cos(n\arccos x) \end{align*} $$ and $$ \begin{align*} T_0(x)={} & 1, \\ T_1(x)={} & x, \\ T_{n+1}(x)={} & 2xT_n(x)-T_{n-1}(x). \end{align*} $$ Show that these formulas are equivalent.
Proof. First we prove that the trigonometric form satisfies the recurrence relation. Clearly, we have that $$ \begin{align*} T_0(x)=\cos0=1,\qquad T_1(x)=\cos(\arccos x)=x. \end{align*} $$ Moreover, we can compute that $$ \begin{align*} T_{n+1}(x)={} & \cos((n+1)\arccos x)=\cos(\arccos x+n\arccos x) \nonumber\\ ={} & \cos(\arccos x)\cos(n\arccos x)-\sin(\arccos x)\sin(n\arccos x) \nonumber\\ ={} & 2\cos(\arccos x)\cos(n\arccos x)-(\sin(\arccos x)\sin(n\arccos x)+\cos(\arccos x)\cos(n\arccos x)) \nonumber\\ ={} & 2\cos(\arccos x)\cos(n\arccos x)-\cos(n\arccos x-\arccos x) \nonumber\\ ={} & 2\cos(\arccos x)\cos(n\arccos x)-\cos((n-1)\arccos x)=2xT_n(x)-T_{n-1}(x). \end{align*} $$ Now we prove the the recurrence relation yields the trigonometric form by strong induction. The base cases $n=0$ and $n=1$ clearly hold. Now we assume that $T_k(x)=\cos(k\arccos x)$ for all $k\leq n$. Then for $k=n+1$, we can use the inductive hypothesis and compute that $$ \begin{align*} T_{n+1}(x)={} & 2xT_n(x)-T_{n-1}(x)=2x\cos(n\arccos x)-\cos((n-1)\arccos x) \nonumber\\ ={} & 2x\cos(n\arccos x)-\cos(n\arccos x-\arccos x) \nonumber\\ ={} & 2x\cos(n\arccos x)-(\cos(n\arccos x)\cos(\arccos x)+\sin(n\arccos x)\sin(\arccos x)) \nonumber\\ ={} & 2x\cos(n\arccos x)-x\cos(n\arccos x)-\sin(n\arccos x)\sin(\arccos x) \nonumber\\ ={} & x\cos(n\arccos x)-\sin(n\arccos x)\sin(\arccos x) \nonumber\\ ={} & \cos(\arccos x)\cos(n\arccos x)-\sin(n\arccos x)\sin(\arccos x) \nonumber\\ ={} & \cos(\arccos x+n\arccos x)=\cos((n+1)\arccos x), \end{align*} $$ so our inductive assumption also holds for $k=n+1$. Therefore, we can conclude that the recurrence relation yields the trigonometric form. We have shown the equivalence between the two forms, so the proof is complete.

Let $\mathcal{P}_n$ be the space of functions defined on $[-1,1]$ that can be described by polynomials of degree less than or equal to $n$ with coefficients in $\RR$. (a) Show that for pairwise distinct points $x_0,x_1,\cdots,x_n\in[-1,1]$, the Lagrange polynomials $L_k(x)$ are in $\mathcal{P}_n$, and that they are linearly independent. Note that this implies that the $n+1$ Lagrange polynomials also form a basis of $\mathcal{P}_n$.
Proof. Note that each Lagrange polynomial can be written as $$ \begin{align*} L_k(x)=\prod_{i=0,i\neq k}^n\frac{x-x_i}{x_k-x_i}. \end{align*} $$ Clearly, each $L_k$ is of order at $n$ since $x_i$ are pairwise distinct, so that $L_k\in\mathcal{P}_n$. Moreover, if for some constants $c_0,\cdots,c_n$, we have that $$ \begin{align*} \sum_{k=0}^nc_kL_k(x)=0,\qquad\forall x\in[-1,1]. \end{align*} $$ Specifically, if we evaluate as each $x=x_i$, we have that $$ \begin{align*} \sum_{k=0}^nc_kL_k(x_i)=\sum_{k=0}^nc_k\delta_{ik}=c_i=0. \end{align*} $$ Therefore, all the constants $c_0,\cdots,c_n$ must be zero, implying that the Lagrange polynomials $L_0,\cdots,L_n$ are linearly independent. Moreover, there are $n+1$ of them, which is equal to the dimension of $\mathcal{P}_n$, so that the Lagrange polynomials form a basis of $\mathcal{P}_n$, and the proof is complete.
(b) Since both the monomials and the Lagrange polynomials are a basis of $\mathcal{P}_n$, each $p\in\mathcal{P}_n$ can be written as linear combination of monomials as well as Lagrange polynomials, i.e., $$ \begin{align*} p(x)=\sum_{k=0}^nb_kL_k(x)=\sum_{k=0}^na_kx^k, \end{align*} $$ with appropriate coefficients $a_k,b_k\in\RR$. Note that $$ \begin{align*} f(x_i)=p(x_i)=\sum_{k=0}^na_kx_i^k= \begin{pmatrix} 1 & x_i & \cdots & x_i^n \end{pmatrix} \begin{pmatrix} a_0 \\ a_1 \\ \vdots \\ a_n \end{pmatrix}. \end{align*} $$ Expanding the matrix expression, we can write that $$ \begin{align*} \underbrace{\begin{pmatrix} 1 & x_0 & \cdots & x_0^n \\ 1 & x_1 & \cdots & x_1^n \\ \vdots & \vdots & \ddots & \vdots \\ 1 & x_n & \cdots & x_n^n \end{pmatrix}}_{V} \underbrace{\begin{pmatrix} a_0 \\ a_1 \\ \vdots \\ a_n \end{pmatrix}}_{\mathbf{a}}= \begin{pmatrix} f(x_0) \\ f(x_1) \\ \vdots \\ f(x_n) \end{pmatrix}. \end{align*} $$ Furthermore, note that the interpolation polynomial can be written in the standard Lagrange form as $$ \begin{align*} p(x)=\sum_{k=0}^nL_k(x)f(x_k)= \begin{pmatrix} L_0(x) & L_1(x) & \cdots & L_n(x) \end{pmatrix} \underbrace{\begin{pmatrix} f(x_0) \\ f(x_1) \\ \vdots \\ f(x_n) \end{pmatrix}}_{\mathbf{b}}, \end{align*} $$ with the Lagrange polynomials $L_k(x)$ as its basis.
