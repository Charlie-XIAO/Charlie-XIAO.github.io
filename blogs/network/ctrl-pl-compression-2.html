You can recall or check those concepts in my previous post Control Plane Compression I.

The theorems, lemmas, and proofs in this post are written in my own version, but the concepts and some ideas in the proofs are attributed to the original paper Control Plane Compression[1] and its extended version which can be found in its reference list.

Utility Definitions

CP-equivalence is powerful, because it can preserve various properties between the concrete network and the abstract network. These properties form our utility definitions as follows:

  • Reachability is preserved if the following holds for any pair of nodes $u,v\in V$: $v$ is reachable from $u$ if and only if $f(v)$ is reachable from $f(u)$.
  • Path lengths are preserved if the following holds for any pair of nodes $u,v\in V$: all paths between $u$ and $v$ have length $n$ if and only if all paths between $f(u)$ and $f(v)$ have length $n$.
  • Black holes are preserved if the following holds for any path $s\subseteq V$: $s$ ends with $\perp$ if and only if $f(s)$ ends with $\perp$.
  • Multipath consistency is preserved if the following holds for any pair of nodes $u,v\in V$: traffic sent from $u$ is reachable along some path to $v$ but dropped along another path if and only if traffic sent from $f(u)$ is reachable along some path to $f(v)$ but dropped along another path.
  • Waypointing is preserved if the following holds for any set of nodes $w_1,\cdots,w_n\in V$: traffic will be waypointed through some $w_i$ if and only if traffic will be waypointed through some $f(w_j)$.
  • Routing loops are preserved if the following holds: there is a routing loop in the abstract network if and only if there is a routing loop in the concrete network.



Proofs

In this section, we will prove that the utilities above are preserved given CP-equivalence.

Theorem 1. Let $SRP$ and $\widehat{SRP}$ be CP-equivalent, then each node $w_1$ in the concrete network forwards along a path $p=(w_1,\cdots,w_n)$ to some node $w_n$ with $f(w_i)=\widehat{w_i}$, $i=1,\cdots,n$, if and only if $\widehat{w_1}$ forwards along a path $\widehat{p}=(\widehat{w_1},\cdots,\widehat{w_n})$ to some node $\widehat{w_n}$ in the abstract network.

Proof of ($\Rightarrow$). Let $p=(w_1,\cdots,w_n)$ be an arbitrary forwarding path in the concrete network such that $f(w_i)=\widehat{w_i}$, $i=1,\cdots,n$, and let $\mathcal{L}$ be a solution to $SRP$. Then we have that $$ (w_i,w_{i+1})\in\texttt{fwd}_{\mathcal{L}}(w_i),\qquad\forall i\in\{1,\cdots,n-1\}. $$ Since $SRP$ and $\widehat{SRP}$ are CP-equivalent, there exists a solution $\widehat{\mathcal{L}}$ to $\widehat{SRP}$, such that $\mathcal{L}$ and $\widehat{\mathcal{L}}$ are forwarding-equivalent, i.e., $$ \forall u,v\in V:\;(u,v)\in\texttt{fwd}_{\mathcal{L}}(u)\iff(f(u),f(v))\in\texttt{fwd}_{\widehat{\mathcal{L}}}(f(u)). $$ Using the $\Rightarrow$ direction of the statement above, we can see that $$ (f(w_i),f(w_{i+1}))=(\widehat{w_i},\widehat{w_{i+1}})\in\texttt{fwd}_{\widehat{\mathcal{L}}}(f(w_i)),\qquad\forall i\in\{1,\cdots,n-1\}. $$ Therefore, $\widehat{w_1}$ forwards along the path $\widehat{p}=(\widehat{w_1},\cdots,\widehat{w_n})$ to $\widehat{w_n}$ in the abstract network. The proof is thus complete by arbitrariness of $p$.   Q.E.D.

Proof of ($\Leftarrow$). Let $\widehat{p}=(\widehat{w_1},\cdots,\widehat{w_n})$ be an arbitrary forwarding path in the abstract network, and let $\widehat{\mathcal{L}}$ be a solution to $\widehat{SRP}$. Then we have that $$ (\widehat{w_i},\widehat{w_{i+1}})\in\texttt{fwd}_{\widehat{\mathcal{L}}}(\widehat{w_i}),\qquad\forall i\in\{1,\cdots,n-1\}. $$ Since $SRP$ and $\widehat{SRP}$ are CP-equivalent, there exists a solution $\widehat{\mathcal{L}}$ to $\widehat{SRP}$, such that $\mathcal{L}$ and $\widehat{\mathcal{L}}$ are forwarding-equivalent, i.e., $$ \forall u,v\in V:\;(u,v)\in\texttt{fwd}_{\mathcal{L}}(u)\iff(f(u),f(v))\in\texttt{fwd}_{\widehat{\mathcal{L}}}(f(u)). $$ Using the $\Leftarrow$ direction of the statement above, for some $\{w_i\}_{i=1}^n$ such that $f(w_i)=\widehat{w_i}$, $\forall i=1,\cdots,n$, we can see that $$ (w_i,w_{i+1})\in\texttt{fwd}_{\mathcal{L}}(w_i),\qquad\forall i\in\{1,\cdots,n-1\}. $$ Therefore, $w_1$ forwards along the path $p=(w_1,\cdots,w_n)$ to $w_n$ in the concrete network. The proof is thus complete by arbitrariness of $\widehat{p}$.   Q.E.D.

Lemma 1. If $SRP$ and $\widehat{SRP}$ are CP-equivalent, then reachability is preserved.

Proof. To see that reachability is preserved, we need to show for any pair of nodes $u,v\in V$ that $v$ is reachable from $u$ if and only if $f(v)$ is reachable from $f(u)$. Fix arbitrary $u$ and $v$ and assume that $v$ is reachable from $u$, then there exists a forwarding path $p=(u,w_1,\cdots,w_k,v)$ in the concrete network. By Theorem 1, we know that there also exists a forwarding path $\widehat{p}=(f(u),f(w_1),\cdots,f(w_k),f(v))$ in the abstract network, thus proving that $f(v)$ is reachable from $f(u)$. By arbitrariness of $u$ and $v$, we can thus conclude the only if direction. The if direction is symmetric since Theorem 1 is an if and only if statement, so the proof is complete.   Q.E.D.

Lemma 2. If $SRP$ and $\widehat{SRP}$ are CP-equivalent, then path lengths are preserved.

Proof. To see that path lengths are preserved, we need to show for any pair of nodes $u,v\in V$ that all forwarding paths between $u$ and $v$ are of length $n$ if and only if all forwaring paths between $f(u)$ and $f(v)$ are of length $n$. This is equivalent to showing the contrapositive statements, that is, there exists a forwarding path $\widehat{p}$ between $f(u)$ and $f(v)$ not of length $n$ if and only if there exists a forwarding path $p$ between $u$ and $v$ not of length $n$. Fix an arbitrary path forwarding $p$ of not of length $n$ in the concrete network, say $p=(w_1,\cdots,w_k)$ with $k\neq n$. By Theorem 1, we know that there also exists a forwarding path $\widehat{p}=(f(w_1),\cdots,f(w_k))$ in the abstract network. Clearly, it is of length $k\neq n$, so the if direction is done by arbitrariness of $p$. The only if direction is symmetric since Theorem 1 is an if and only if statement, so the proof is complete.   Q.E.D.

Theorem 2. Let $SRP$ and $\widehat{SRP}$ be CP-equivalent, and let $\mathcal{L}$ and $\widehat{\mathcal{L}}$ be the solutions to $SRP$ and $\widehat{SRP}$, respectively. Then each node $w_1$ in the concrete network forwards along a label path $s=(\mathcal{L}(w_1),\cdots,\mathcal{L}(w_n))$ to some node $w_n$ with $f(w_i)=\widehat{w_i}$, $i=1,\cdots,n$, if and only if $\widehat{w_1}$ forwards along the label path $\widehat{s}=(\widehat{\mathcal{L}}(\widehat{w_1}),\cdots,\widehat{\mathcal{L}}(\widehat{w_n}))$ to $\widehat{w_n}$ and $h(s)=\widehat{s}$.

Proof. By Theorem 1, if any $w_1$ forwards along the label path $s=(\mathcal{L}(w_1),\cdots,\mathcal{L}(w_n))$ to some $w_n$, and $f(w_i)=\widehat{w_i}$, $i=1,\cdots,n$, $\widehat{w_1}$ must forward along the label path $\widehat{s}=(\widehat{\mathcal{L}}(\widehat{w_1}),\cdots,\widehat{\mathcal{L}}(\widehat{w_n}))$ to $\widehat{w_n}$. Therefore, it suffices to show that $h(s)=\widehat{s}$. Indeed, since $SRP$ and $\widehat{SRP}$ are CP-equivalent, the solutions $\mathcal{L}$ and $\widehat{\mathcal{L}}$ are label-equivalent, i.e., $h(\mathcal{L}(w_i))=\widehat{\mathcal{L}}(f(w_i))=\widehat{\mathcal{L}}(\widehat{w_i})$ for all $w_i$. Therefore, we can see that $$ \widehat{s}=(\widehat{\mathcal{L}}(\widehat{w_1}),\cdots,\widehat{\mathcal{L}}(\widehat{w_n}))=(h(\mathcal{L}(w_1)),\cdots,h(\mathcal{L}(w_n)))=h(s), $$ which concludes the only if direction of the theorem by arbitrariness of the path. The if direction is symmetric since both Theorem 1 and label-equivalence are if and only if statements, so the proof is complete.   Q.E.D.

Lemma 3. If $SRP$ and $\widehat{SRP}$ are CP-equivalent, then black holes are preserved.

Proof. To see that black holes are preserved, we need to show for any path $p$, it ends with $\perp$ if and only if $f(p)$ ends with $\perp$. This is immediate by Theorem 2 and by drop-equivalence, which states that for any attribute $a$, $a=\perp$ if and only if $h(a)=\perp$. The proof is thus complete.   Q.E.D.

Lemma 4. If $SRP$ and $\widehat{SRP}$ are CP-equivalent, then multipath consistency is preserved.

Proof. To see that multipath consistency is preserved, we need to show that for any node $u\in V$, traffic sent from any node $u$ with $f(u)=\widehat{u}$ is reachable along some path to $v$ with $f(v)=\widehat{v}$ but dropped along some other path if and only if traffic sent from $\widehat{u}$ is reachable along some path to $\widehat{v}$ but dropped along some other path. Assume that there is a forwarding path $p=(u,w_1,\cdots,w_k,v)$ with $f(u)=\widehat{u}$ and $f(v)=\widehat{v}$, yet another path $p'$ starting from $q$ that ends with $\perp$. The assumption above means that traffic sent from $u$ will be forwarded to $v$ successfully along $p$ but dropped at along $p'$. By Theorem 1, there must also be a forwarding path $\hat{p}=(\widehat{u},f(w_1),\cdots,f(w_k),\widehat{v})$, so indeed traffic sent from $\widehat{u}$ is reachable along the forwarding path $\hat{p}$ to $\widehat{v}$. Moreover, by Lemma 3, black holes must be preserved, so $f(p')$ also ends with $\perp$. Note that $f(p')$ starts from $f(u)=\widehat{u}$, so we have also found a path along which traffic sent from $\widehat{u}$ will be dropped. By arbitrariness of $u$, we can conclude the only if direction of the lemma. The if direction is symmetric since both Theorem 1 and Lemma 3 are if and only if statements, so the proof is complete.   Q.E.D.



Summary

In this post, I have discussed the first part of the paper. This includes some important notations and the definition of CP-equivalence, which can be used to prove various utility preservation. However, CP-equivalence is hard to evaluate, so the authors have furthermore proposed conditions for effective abstractions, which can sufficiently imply CP-equivalence. If there were to be more posts on this paper, I will discuss some important utility definitions and how CP-equivalence can guarantee them. I may not discuss the proofs of how effective abstractions can guarantee CP-equivalence. At last, thanks for your meticulous reading and hope you enjoy it.
